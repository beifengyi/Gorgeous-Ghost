Privacy Policy

Last Updated: January 2024

Data Collection

"Gorgeous Ghost" respects user privacy. We are committed to protecting your personal information. As a single-player game, this application collects only the following necessary data for gameplay:

  • User's level ID (solely for saving game progress, enabling continuation from where the user left off)

No other data (including but not limited to personal identification information, device details, or usage behavior) is collected, stored, or transmitted by the game.

Data Usage

The collected data is used exclusively for one purpose:

  • To save the user's game progress via level ID, ensuring the ability to resume gameplay from the last level reached

Data Protection

We implement appropriate security measures to protect your data:

  • All data (level ID) is stored locally on the user's device only
  • No data is uploaded to cloud servers or shared with any third parties
  • No personal information is collected or processed at any stage

Meta Quest Platform Data

Please note that the Meta Quest platform may independently collect certain data related to the game, such as payment status, retention rates, and activity metrics. This data collection is conducted solely by Meta Quest as part of its platform services.

We (the developer of "Gorgeous Ghost") have no access to, control over, or responsibility for the data collected by the Meta Quest platform. Such data handling is governed by Meta's own privacy policy.

User Rights

You have the right to:

  • Be informed of the limited data we collect (as stated herein)
  • Request the deletion of your locally stored level ID data (by clearing the game's local storage through your device settings)
  • Refuse data collection (note: this may prevent the game from saving progress)
